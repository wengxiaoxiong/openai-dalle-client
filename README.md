# openai-dalle-client
this is a web client for OPENAI picture generator, developed by Vue2 , U can use your API-Key for using this application
## 使用说明
这个不需要部署，你只需要输入openAI的api-key就可以使用了，具体怎么获得api-key自己想办法吧～仅仅只是一个客户端而已。
